Maybe, the simple solution is... don't use it

## Update from 2.x

```sh
cd ~/MagicMirror/modules/MMM-GoogleAssistant
npm run update
```

## Update on new MagicMirror version (exemple v2.10.x to v2.11.x)
```sh
cd ~/MagicMirror/modules/MMM-GoogleAssistant
npm run rebuild
```
